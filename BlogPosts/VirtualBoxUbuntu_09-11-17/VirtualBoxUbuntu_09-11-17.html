Setting up Ubuntu with VirtualBox 09-11-17

VirtualBox is an amazing free program! It allows you set up a virtual computer within your computer that has a different operating system. I use VirtualBox to run an Ubuntu computer on my Windows machine. I need the Ubuntu to run a lot of my analyses - especially those dealing with sequencing data.

Having had to frequently set up Ubuntu using VirtualBox and always running into difficulties, I've written this post to describe how I got it working this time.

1) Download an Ubuntu iso file from (I just used Ubuntu v16.04)here

2) Download and install VirtualBox
   - Download the executable file from here
   - Note most recent version 5.2 didn't work for me. I downloaded an older exexcutable (v5.1.30) file from here

3) Create a new Ubuntu machine
   - Open VirtualBox and select "New"
   - Choose a name e.g. "Ubuntu", leave Type: "Linux" and Version: "Ubuntu (64-bit)". Click "Next"
   - Assign a maximum amount of RAM (Rapid Access Memory) to the machine - at least 4000MB. Click "Next"
   - Leave "Create a virtual hard disk now" selected. Click "Create"
   - Leave "VDI (VirutalBox Disk Image) selected. Click "Next"
   - Leave "Dynamically allocated" selected. Click "Next"
   - Set the hard drive size. I usually assign around 50GB. Click "Create"

4) Set up the Ubuntu machine
   - Select the machine you just built and click "Start"
   - Click on the Folder image on the right and select the Ubuntu iso file from Step 1. Click "Start"
   - Wait a few minutes. Select "Install Ubuntu"
   - Tick both of the options for updates and third party software. Click "Continue"
   - Select "Erase disk and install Ubuntu". Don't panic - this won't remove any files from your computer, it is referring to the virtual computer. Select "Install Now"
   - Select "Continue"
   - Select the country you are based in. Click "Continue"
   - Select the keyboard language you wish to use. Click "Continue"
   - Type your name - ideally without spaces - and password. Click "Continue"
   - Wait for ages... Click "Restart Now"

5) Installing Guest Additions iso file
   - Once the Ubuntu has finished restarting, hover the mouse over "Devices" on the bar at the top left. Select "Insert Guest Additions CD image..."
   - Select "Run"
   - Type in your password (the one you set a few minutes ago). Click "Authenticate"
   - Wait for the installation to finish. If there are no errors, press the Enter key to finish. If there were errors try searching online for a solution
   - Click the red x in top right corner and select "Power off the machine" to shutdown the machine and close the window

6) Setting up the Shared Folder
   - Create a folder to share with the Ubuntu machine on your Windows machine
   - In VirtualBox select your machine and click on "Settings"
   - Select "Shared Folders" on the left. Click on the Folder icon with the plus sign on the right hand side
   - In "Folder Path" drop down menu, select "Other" and then select the folder you just created on your Windows machine
   - Tick "Auto-mount" and leave "Read-only" unticked. Click "OK"
   - Start up your Ubuntu machine again by selecting the machine and clicking "Start"
   - Once the machine has started up, click on the swirly symbol in the top left corner and type in "terminal" into the search bar. Click on "Terminal" application
   - Type the following command into the terminal:

$ sudo usermod -a -G vboxsf [Username]

   - Restart the Ubuntu machine
   - Navigate to "/media/" and check if a folder called "sf_[YourSharedFolderName]" is present there or in a sub directory
   - If the folder is present - SUCCESS! - type the following to create a shortcut to the folder on the Ubuntu desktop:

$ sudo ln -s /media/sf_[YourSharedFolderName] /home/[YourUserName]/Desktop/[YourSharedFolderName]

That's it done! Hopefully all of the above worked - installing the Guest Additions and getting the Shared Folder to work can be a bit temperamental!
