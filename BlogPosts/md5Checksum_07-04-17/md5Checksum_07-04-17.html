What is a file's md5sum?

A file's md5sum is a unique hash key (sequence of letters and numbers). This key is useful because it identifies the file contents and will change if the file contents is changed in any way.

I have used md5sum file keys when I have needed to transfer files. A set keys for the files to be transferred is generated on the source and sink computers. If the md5sum keys created before and after a file transfer are identical then the file transfer has been successful.

To generate an md5sum key for a file, use the following code:

# On a UNIX machine
$ md5sum [file name]

# On a Mac OS X
$ md5 [file name]
