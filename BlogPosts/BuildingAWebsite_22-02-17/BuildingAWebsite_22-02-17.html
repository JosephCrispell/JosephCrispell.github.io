Building my website

I created this website to provide a little insight into my interests and research. My hope is that it will be a useful resource for me and others.

This website is built using a combination of html, css and javascript. Part of my motivations for building this site was to prompt me to learn a little about these languages. I have taken my instruction from a large number of online resources but the following were the most useful:
-   w3schools
-   stackoverflow
-   tutorialspoint

To get started first create a respository with the name: [username].github.io. Once created you'll need to clone the respository on your computer. On a UNIX computer use:

# Clone repository
$ git clone https://[username]:[password]@github.com/[username]/[username].github.io

# Configure username and password
$ git config --global user.name "[username]"
$ git config --global user.email [email]

With the cloned respository you'll be able to add and edit new html pages. Push any changes to github using:

# Add new file to repository
$ git add [filename]

# Or add all files and folders in directory
$ git add *

# Remove any deleted files from repository
$ git add . -A

# Commit changes to repository
$ git commit -m "[Message referring to commit]"

# Push commit to repository
$ git push

I have created a simple bash script that can be ran in the UNIX command line for syncing the changes made on a local repository to gihub. It is available on github.
