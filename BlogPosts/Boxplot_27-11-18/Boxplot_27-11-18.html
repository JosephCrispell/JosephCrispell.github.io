A boxplot in R 03-12-18

A wee while ago I gave a lecture about summarising data. One commonly used type of plot for summarising data is a boxplot. I spent a bit of time figuring out what the boxplot actually represents in R.

After dealving through the ?boxplot() documentation and loads of searching online, I think I've got a grasp of it. So lets create a boxplot:

# Generate some example points
values <- rexp(n=150, rate=2)

# Create a boxplot of the values
boxplot(values, xlab="", ylab="", yaxt="n", xaxt="n", frame=F, las=1, pch=19)

I purposely removed the axes and I've added some arrows and labels using the arrows() and text() functions. The boxplot illustrates a few things:

  • Median - A measure of the averageness of the data
  • Quartile - The input data (values) are ordered and split into four equal sized groups and their boundaries are marked by the quartiles: 25% (1st quartile), 50% (2nd), 75% (3rd), 100% (4th). The box of the boxplot ranges from the 1st to the 4th quartile. Quartiles can be calculated in R using the quantile() function.
  • Fence - These represent the outer range (minimum and maximum) boundaries of the data provided (values), excluding any outliers
  • Outlier - A data point that is very distinct from the rest

The tricky part of the way R creates a boxplot is how does it identify outliers? R uses the inter-quartile range (IQR): 75% quartile - 25% quartile.

The fences span between the minimum and maximum numbers that are within the median -/+ 1.5*IQR. Any values outside this range are labelled outliers.

The value of 1.5 is the default value and can be changed using the range parameter of the boxplot() function.

So that's the boxplot - a useful way to summarise your data. Just a note, I often like to overlay my boxplots with the original datapoints using the stripchart() function.
