A quick post about my new R function: plotMultipleHistograms() 07-02-19

Today I lost myself in creating a new R function called plotMultipleHistograms(), in my basicPlotteR package. This function plots multiple histograms on the same plot.

Histograms are a type of plot I use a lot, they summarise the shape of a distribution. Each bar tells me how many values fell within that particular bin.

Sometimes I like to compare multiple distributions, and I usually plot multiple histograms to do this. The plotMultipleHistograms() helps you to plot multiple histograms in the same plot, here it is in action:

# Install and load the devtools package
install.packages("devtools")

# Install and load the basicPlotteR package
devtools::install_github("JosephCrispell/basicPlotteR")
library(basicPlotteR)

# Set the seed
set.seed(254534)

# Create three random samples from a normal distribution
distributions <- list(rnorm(500, mean=5, sd=0.5),
  rnorm(500, mean=8, sd=5),
  rnorm(500, mean=20, sd=2))

# Plot overlapping histograms
plotMultipleHistograms(distributions, nBins=20,
  colours=c(rgb(1,0,0, 0.5),
  rgb(0,0,1, 0.5), rgb(0,1,0, 0.5)), las=1,
  main="Samples from normal distributions",
  xlab="Value")

plotMultipleHistograms() uses the following steps:

  1. Calculate the minimum and maximum values across the distributions - X axis limits
  2. Using the X axis limits, define the breaks of each of the bins used to plot each distribution
  3. Create each histogram object but don't plot it
  4. Find the maximum bin count across the histograms - Y axis limit
  5. Plot each histogram within the defined Y axis limits
